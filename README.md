# Momentum

Momentum es un proyecto construido con HTML, CSS, JavaScript y PHP que permite gestionar tus trucos de skate. El repositorio está organizado en carpetas como php/, wall/, img/ y otras, lo que sugiere que incluye funcionalidad de autenticación, muro, etc.
